Rust Is Broken

On Mac OS X

Days and Counting

Rust is currently in an unplayable state due to a bug introduced in the Mac build of Unity 2019.3, released Jan 27, 2020. Both FacePunch and Unity are AWARE of this issue and have done nothing to resolve it in over a year. The game is still being advertised and sold to Mac users at full-price despite it's broken state.
